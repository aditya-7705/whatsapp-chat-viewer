# WhatsApp Chat Viewer

A web application that converts exported WhatsApp chats (.txt or .zip) into a clean, WhatsApp-like interface.

## Features

* Upload WhatsApp exported .txt files
* Upload WhatsApp exported .zip files
* WhatsApp-style chat interface
* Message search
* Date-based navigation
* Sent and received message statistics
* Local processing for privacy
* Mobile-friendly design

## How It Works

1. Export a WhatsApp chat.
2. Upload the exported .txt or .zip file.
3. View the conversation in a familiar chat interface.
4. Search messages and browse by date.
5. See message statistics instantly.

## Privacy

All chat processing happens locally in the browser. Chat data is not stored on any server.
